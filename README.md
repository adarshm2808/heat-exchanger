
Designed a shell-and-tube heat exchanger model using SolidWorks to demonstrate fluid flow and thermal exchange concepts. Created a 3D assembly with inlet/outlet connections and internal tube arrangement for mechanical and industrial application visualization

## Overview

This project focuses on the design and 3D modeling of a shell-and-tube heat exchanger using SolidWorks. The model demonstrates the structural arrangement used for heat transfer between two fluids in industrial systems.

The project was developed to understand thermal equipment design, cylindrical assemblies, and piping integration in mechanical systems.


## Features

* 3D shell-and-tube heat exchanger model
* Cylindrical outer shell structure
* Inlet and outlet pipe connections
* Internal tube arrangement visualization
* Industrial equipment-oriented design
* Realistic rendered CAD model

## Components Modeled

* Outer Shell
* Internal Tube Section
* Inlet Pipe
* Outlet Pipe
* End Covers (lids)



## Software Used

* SolidWorks



## Design Process

1. Designed the cylindrical shell structure.
2. Created internal tube section for heat transfer representation.
3. Added inlet and outlet pipe connections.
4. Developed assembly alignment and geometric structure.
5. Applied rendering for visualization and presentation.
